# Release Notes

## 版本主题
多直播同屏第二轮增强（双主布局优化 + 自由布局 + 真实全屏交互完善）。

## 版本信息
- Tag: v2.0.14+25-multilive-20260325
- 平台重点: Windows（个人自用优先）

## 主要更新

### 1. 新增自由布局模式（随意排列）
- 在同屏布局中新增“自由布局”模式。
- 每个窗格支持直接拖拽移动。
- 支持窗格右下角拖拽缩放。
- 增加磁吸对齐（贴边/贴邻近窗格自动吸附），更容易排出自定义阵列。

### 2. 2主多副布局方向优化
- 2主多副调整为“左侧双主纵向堆叠 + 右侧副屏网格”。
- 双主与副屏分区更稳定，便于同时关注两个主直播间。

### 3. 全屏交互升级（防误触）
- 同屏全屏默认隐藏工具栏，鼠标悬停时显示。
- 提供右侧悬浮退出按钮。
- 同屏全屏不使用 Esc 退出，避免误触中断。

### 4. 容量策略与交互延续
- 同屏容量继续采用“自动识别 + 性能优先”策略。
- 取消编辑模式后，保持“直接拖拽重排”的快捷交互。
