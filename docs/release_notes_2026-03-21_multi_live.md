# Release Notes

## 版本主题
多直播同屏体验升级（真实全屏 + 自动容量 + 交互简化）。

## 主要更新

### 1. 真实全屏体验（Windows 优先）
- 同屏页新增真实全屏入口，进入后覆盖任务栏。
- 全屏状态下隐藏页面顶部区域，仅在顶部悬停时下拉显示工具栏。
- 提供右侧悬浮退出全屏按钮。
- 多直播全屏流程不再使用 Esc 退出，避免误触退出。

### 2. 同屏布局自适应升级
- 布局能力覆盖 1-16 窗格的规则化自适应。
- 4 窗格在 1主多副模式固定为：左侧 1 主 + 右侧 3 个纵向副屏。
- 主副布局继续支持主屏比例调节与分割条拖动。

### 3. 自动容量识别（性能优先）
- 移除布局面板中的手动同屏上限滑杆。
- 上限改为按当前窗口逻辑尺寸自动估算，并施加平台硬上限。
- 当前策略为性能优先：
  - 桌面端：阈值更保守，自动上限硬顶 12。
  - 移动端：阈值更保守，自动上限硬顶 6。

### 4. 交互简化
- 取消编辑模式，改为默认直接拖拽重排。
- 快捷键说明同步更新：真实全屏进入/退出使用工具栏按钮与 F，不再宣传 Esc 退出路径。
