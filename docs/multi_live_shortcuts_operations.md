# 同屏播放快捷键与操作手册（当前版本）

## 1. 构建结果

- Windows Release 构建已完成。
- 可执行文件：build/windows/x64/runner/Release/pure_live.exe

## 2. 快捷键总表（同屏页面）

> 生效页面：同屏播放页面（MultiLivePage）

| 快捷键 | 功能 | 说明 |
|---|---|---|
| 1-9 | 切换焦点窗格 | 将对应序号窗格设为焦点窗格（存在才生效） |
| M | 静音焦点窗格 | 对当前焦点窗格执行静音/取消静音 |
| 0 | 静音焦点窗格 | 与 M 等效（toggle mute） |
| R | 刷新焦点窗格 | 重新拉流当前焦点窗格 |
| Delete | 删除焦点窗格 | 删除后可在 3 秒内撤销 |
| F | 全屏/退出全屏（焦点窗格） | 让焦点窗格进入沉浸全屏，再次按 F 可退出 |
| = | 焦点窗格音量 +5% | 单窗音量快捷调整 |
| - | 焦点窗格音量 -5% | 单窗音量快捷调整 |
| Shift + = | 总音量 +5% | 调整同屏全局音量 |
| Shift + - | 总音量 -5% | 调整同屏全局音量 |

## 3. 房间卡片入口操作

在首页/列表的房间卡片上：

- 队列按钮：加入/移出同屏队列（按钮会显示“已在队列”状态）。
- 同屏按钮：立即打开同屏播放；若当前房间不在队列，会自动附加当前房间。
- 点击卡片主体：进入单房间播放页（原有行为）。
- 长按卡片：查看房间信息与关注按钮（原有行为）。

## 4. 同屏页面顶部工具栏

### 4.1 基础功能

- 添加：打开三标签添加面板。
- 队列(N)：将同屏队列中的房间批量加入当前同屏。
- 全静音：所有窗格静音。
- 全刷新：刷新所有窗格（会二次确认）。
- 布局：打开布局设置面板。

### 4.2 音频功能

- 总音量滑杆：统一控制所有窗格输出总增益。
- 音频模式切换：
  - 单焦点：仅焦点窗格出声。
  - 混音：所有未静音窗格都可出声。
  - 主窗优先：主窗全音量，其余窗格按较低权重出声。

## 5. 同屏页面右上角“更多”菜单

- 队列策略: 保持
  - 从队列添加后，不改变队列内容。
- 队列策略: 移除已添加
  - 成功加入同屏的房间会从队列移除。
- 队列策略: 全清空
  - 一次添加后清空队列。
- 清空同屏队列
  - 手动清空队列（会二次确认）。
- 打开设置
  - 跳转设置页面。
- 快捷键帮助
  - 弹窗显示核心快捷键说明。
- 关于同屏
  - 弹窗显示同屏功能简介。

## 6. 窗格（Tile）内操作

鼠标移入窗格顶部控制条后可用：

- 清晰度：切换清晰度档位。
- 线路：切换播放线路。
- 静音：单窗静音开关。
- 音量调节：弹窗滑杆设置该窗格音量。
- 刷新：重试当前窗格。
- 移除：删除当前窗格（支持撤销）。
- 全屏：该窗格进入/退出全屏。

补充交互：

- 点击任意窗格：设为焦点。
- 1主多副布局下点击副窗：升为主窗。
- 直接拖拽窗格：重排顺序（无需编辑模式）。

## 7. 布局操作

布局面板支持：

- 布局模式：
  - 等分布局
  - 1主多副
  - 2主多副
- 同屏数量上限：自动识别当前窗口可承载数量（性能优先）
- 主屏比例：30%-70%（主副布局可调）

拖动分割条：

- 1主多副：水平拖动主副分割条，实时调整主屏宽度比例。
- 2主多副：垂直拖动主副分割条，实时调整主屏高度比例。

## 8. 全屏沉浸模式操作

- 进入方式：
  - 按 F（焦点窗格）
  - 点击窗格顶部“全屏”按钮
- 全屏效果：
-  进入真实全屏（覆盖任务栏），隐藏页面顶部栏和工具栏。
-  顶部悬停热区可唤起控制栏；离开后自动收起。
-  右侧显示悬浮“退出全屏”按钮，移动鼠标可重新唤起。
- 退出方式：
  - 按 F
-  点击右侧悬浮退出按钮

## 9. 删除撤销机制

- 通过 Delete 或窗格关闭按钮移除后，会出现底部 SnackBar。
- 3 秒内点击“撤销”可恢复该窗格（尽量恢复原位置和焦点语义）。
- 若最后一个窗格被移除且未撤销，会自动返回上一页。

## 10. 常见问题

### Q1：我更新了代码但界面看起来没变化

常见原因是启动了旧产物或旧进程。建议：

- 重新执行 Windows Release 构建。
- 从最新产物路径启动：build/windows/x64/runner/Release/pure_live.exe。
- 关闭旧实例后再启动新构建。

### Q2：为什么点了“队列(N)”没有新增

可能原因：

- 队列为空。
- 队列房间已全部在同屏中。
- 已达到同屏上限。

### Q3：为什么有的窗格没声音

先检查：

- 是否处于“单焦点”模式（非焦点窗格默认静音输出）。
- 该窗格是否被手动静音。
- 总音量或单窗音量是否过低。
