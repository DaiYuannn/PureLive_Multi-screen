## 变更摘要
本 PR 聚焦多直播同屏体验升级，完成真实全屏、布局自适应与自动容量识别，并同步文档。

## 主要改动
- 真实全屏
  - 新增窗口级全屏切换状态与控制方法。
  - 全屏下隐藏常驻顶部区域，改为顶部悬停唤起工具栏。
  - 工具栏提供显式退出全屏按钮。
  - 多直播路径关闭 Esc 监听，避免误退出。

- 布局与容量
  - 新增 1-16 窗格布局规则映射。
  - 固化 4 窗格在 1主多副下的左主右三副（纵排）形态。
  - 移除手动同屏上限滑杆，改为窗口尺寸自动估算 + 平台硬上限。
  - 策略默认性能优先（桌面硬顶 12，移动硬顶 6）。

- 交互
  - 取消编辑模式开关，默认支持拖拽重排。
  - 快捷键帮助文案更新，真实全屏入口/出口说明与当前实现一致。

## 测试与验证
- 手工验证
  - 多直播页面进入/退出真实全屏。
  - 全屏状态顶部悬停工具栏显隐。
  - 4 窗格 1主多副布局固定形态。
  - 自动容量上限随窗口变化。
  - 默认拖拽重排与焦点音频策略。

## 风险与回滚
- 风险点
  - 不同平台窗口全屏能力差异。
  - 自动容量估算阈值在极端分辨率下可能需要继续微调。
- 回滚方式
  - 回退本 PR 即可恢复旧交互与容量逻辑。
